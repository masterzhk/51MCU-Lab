# 51MCU-Lab

51单片机实验室，用于学习、练习、测试51单片机。

## 说明

单片机型号：STC89C52  
开发软件：Keil C51  
烧录工具：stc-isp-15xx-v6.86v  
画图工具：立创EDA  

## 构建

*每个步骤的具体操作可以查找对应软件的使用教程了解。*

1. 使用Keil C51打开项目文件*.uvproj，构建/Build生成hex文件。

2. 使用烧录工具把hex文件下载到单片机上。

3. 上电运行。
